# Log Analyzer

Log Analyzer یک برنامه ساده نوشته شده با زبان Go است که فایل‌های لاگ را می‌خواند، تجزیه و تحلیل می‌کند و گزارش‌هایی از خطاها تولید می‌کند. این پروژه برای کمک به شناسایی خطاهای موجود در فایل‌های لاگ استفاده می‌شود.

## ویژگی‌ها

- دریافت فایل لاگ از کاربر.
- خواندن محتویات فایل لاگ.
- تجزیه داده‌ها به خطوط جداگانه.
- شناسایی و استخراج خطاها از لاگ‌ها.
- تولید گزارش از خطاهای شناسایی‌شده.
- نمایش یا ذخیره نتایج تحلیل‌شده.

## نحوه استفاده

### پیش‌نیازها

برای اجرای این برنامه نیاز به نصب Go دارید. برای نصب Go می‌توانید به [وب‌سایت رسمی Go](https://golang.org/dl/) مراجعه کنید و دستورالعمل‌های نصب را دنبال کنید.

### اجرای برنامه

1. ابتدا کد را از مخزن (یا فایل محلی) دریافت کنید.
2. در ترمینال به دایرکتوری پروژه بروید.
3. دستور زیر را برای اجرای برنامه وارد کنید:

   ```bash
   go run log_analyzer.go
